# 📱 Sistema de Agendamento de Chromebooks

Sistema desktop para gerenciamento de empréstimos de chromebooks em instituições de ensino. Desenvolvido em Python com interface gráfica Tkinter.

## ✨ Funcionalidades

- 📊 **Dashboard de disponibilidade** - Visualização em tempo real da quantidade de chromebooks disponíveis
- 📝 **Cadastro de agendamentos** - Registro de retirada com professor, turma, quantidade, data e horários
- 🔍 **Filtros de pesquisa** - Busca por professor e data específica
- 🏫 **Gerenciamento de turmas** - Adicionar, editar, remover e reordenar lista de turmas
- 💾 **Backup automático** - Criação de backups antes de alterações importantes
- 📄 **Exportação de relatórios** - Geração de relatórios em CSV ou Excel
- 🎨 **Interface intuitiva** - Cores indicativas de disponibilidade (verde, laranja, vermelho)
